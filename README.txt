
🚀 Instrucciones para publicar la landing en Netlify

1. ✅ Prepara los archivos
   Reemplaza los archivos vacíos en la carpeta /assets con los reales:
   - logo_spsp.png — logo de la parroquia
   - image3d_p_spsp.png — imagen principal de la iglesia
   - archivodecargaDemo.pdf — archivo descargable con las actividades

2. 📦 Comprime todo el proyecto (si es necesario)
   Asegúrate de tener esta estructura:
   /landing_spsp/
     ├── index.html
     ├── styles.css
     └── /assets/
         ├── logo_spsp.png
         ├── image3d_p_spsp.png
         └── archivodecargaDemo.pdf

3. 🚀 Publicar en Netlify
   a. Ve a: https://app.netlify.com/drop
   b. Arrastra y suelta la carpeta o el archivo ZIP
   c. Netlify generará un enlace público como:
      👉 https://nombre.netlify.app

4. 🌐 (Opcional) Conectar dominio personalizado
   Si deseas un dominio propio como parroquiaspsp.org:
   - Compra el dominio en un proveedor (ej. GoDaddy, Namecheap)
   - En Netlify, ve a "Domain Settings" y sigue el proceso para conectarlo

🛠️ ¿Cómo editar el texto?
Abre el archivo `index.html` con un editor de texto (Visual Studio Code o Bloc de notas).
Puedes modificar los títulos, botón y enlaces directamente desde el HTML.

✉️ Soporte técnico: Consulta con tu desarrollador si necesitas hacer cambios más avanzados.
